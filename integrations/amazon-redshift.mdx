This article guides you through the process of connecting Amazon Redshift to Peaka, allowing you to seamlessly access your Amazon Redshift data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from Amazon Redshift.

- Port
- Host
- User name
- Password
- Database name

Follow the steps below to obtain it:

1. Go to Amazon Redshift.

2. After accessing Amazon Redshift, create your cluster.

<img src="/images/integrations/amazon-redshift-1.png" />

3. After clicking the "Create Cluster" button, you will be redirected to provide some information. Your password and username should match the details shown in the image below.

<img src="/images/integrations/amazon-redshift-2.png" />

4. If you choose to customize the Additional Configurations settings, you have the flexibility to modify your database name. However, if you prefer to stick with the default settings, your database name will be set to "dev" and the port will remain at the default value of 5439. Once you've finished inputting your desired configurations, simply click the "Create Cluster" button located at the bottom of the page. Please be patient as the creation process may take a few minutes to complete.

<img src="/images/integrations/amazon-redshift-3.png" />

5. Accessing your host's knowledge can be done through the "Properties" tab.

<img src="/images/integrations/amazon-redshift-4.png" />

## Connecting Amazon Redshift to Peaka

Now that you have your requirements, you can connect it to Peaka using the following steps:

1. Navigate to the settings page in Peaka.

2. Click the "Add First Connection" button to initiate the connection process.

3. In the modal that appears, select "Amazon Redshift" as your connection type.

4. Enter a name for your connection name and connection details below:

- User name
- Password
- Port
- Host
- You can enable the "Connect through TLS" option if SSL is active.

<img src="/images/integrations/amazon-redshift-5.png" />

5. Click "Next" to create your Amazon Redshift connection.

Congratulations! You've successfully created a connection between Amazon Redshift and Peaka. You can now access and manage your Amazon Redshift data seamlessly within the Peaka platform.
