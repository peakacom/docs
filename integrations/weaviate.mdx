This article guides you through the process of connecting Weaviate to Peaka, allowing you to seamlessly access your Weaviate data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from Weaviate.

- Weaviate Server URL
- API Key

Follow the steps below to obtain it:

1. To obtain Weaviate Server URL and API Key, check this [article](https://weaviate.io/developers/weaviate/quickstart#:~:text=You%20need%20a%20Weaviate%20instance,the%20Details%20tab%20in%20WCD.).

## Connecting Weaviate to Peaka

Now that you have obtained the necessary integration data and credentials, you can connect Weaviate to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. Click the "Bring your data" button to initiate the connection process.

3. In the modal that appears, select "Weaviate" as your connection type.

4. Enter a name for your connection, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/weaviate-1.png" />

5. Click "Authorize" to create your Weaviate connection.

Congratulations! You've successfully connected Weaviate to Peaka. You can now access and manage your Weaviate data seamlessly within the Peaka platform.
