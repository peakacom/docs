This article guides you through the process of connecting Klaviyo to Peaka, allowing you to seamlessly access your Klaviyo data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from Klaviyo.

- API Key

Follow the steps below to obtain it:

1. To obtain the API Key, refer to this [article](https://help.klaviyo.com/hc/en-us/articles/7423954176283).

## Connecting Klaviyo to Peaka

Now that you have obtained the necessary integration data and credentials, you can connect Klaviyo to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Klaviyo" as your data source.

4. Enter a name for your data source, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/klaviyo-1.png" />

5. Click "Sign in" to create your Klaviyo data source.

Congratulations! You've successfully connected Klaviyo to Peaka. You can now access and manage your Klaviyo data seamlessly within the Peaka platform.
