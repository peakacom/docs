This article guides you through the process of connecting Paychex to Peaka, allowing you to seamlessly access your Paychex data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from Paychex.

- **Client ID**
- **Client Secret**

Follow the steps below to obtain it:

Note: For getting your Client Secret, you must obtain it while creating app,

1. Log in to your Paychex account and go into Company Settings.
2. Select Integrated apps.
3. Click Create App.
4. Enter an Application Name and Description.
5. Select Read only for the Company and worker APIs
   - If you also need to sync payroll information, select Read only for the Payroll and check APIs​.
   - If you need access to Payroll data and do not see it as a toggle, contact your account manager for help.
6. Accept the Legal Notice and Paychex Inc, API License Agreement, and Save.
7. You'll then see the API key and Client Secret, which you'll paste into the linking flow

## Connecting Paychex to Peaka

Now that you have obtained the necessary integration data and credentials, you can connect Paychex to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Paychex" as your data source.

4. Enter a name for your data source, fill in the **Client ID** and **Client Secret** fields.

<img
  alt="connect-connector"
  src="https://cdn.peaka.com/integrations/paychex-1.png"
/>

5. Click "Next" to create your Paychex data source.

Congratulations! You've successfully connected Paychex to Peaka. You can now access and manage your Paychex data seamlessly within the Peaka platform.
