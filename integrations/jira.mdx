This article guides you through the process of connecting Jira to Peaka, allowing you to seamlessly access your Jira data within the Peaka platform.

## Prerequisites

Before you start, you will need an API Key from Jira.

Follow the steps below to obtain it:

1. To generate your API Key, please visit the following link: [Atlassian API Key Generation](https://id.atlassian.com/manage-profile/security/api-tokens) and proceed with the creation of your API Key.

<img src="images/integrations/jira-integration-2.png" />

2. To include your service URL, kindly access the following [link](https://id.atlassian.com/manage-profile/link-preferences) and input your service URL in the designated field.

<img src="images/integrations/jira-integration-1.png" />

## Connecting Jira to Peaka

Now that you have your requirements, you can connect it to Peaka using the following steps:

1. Navigate to the settings page in Peaka.

2. Click the "Add First Connection" button to initiate the connection process.

3. In the modal that appears, select "Jira" as your connection type.

4. Enter your connection name and requirements that you obtained and enter user name can be random text.

<img src="images/integrations/jira-integration-3.png" />

5. Click "Authorize" to create your Airtable connection.

<img src="images/integrations/jira-integration-4.png" />

Congratulations! You've successfully connected Jira to Peaka. You can now access and manage your Jira data seamlessly within the Peaka platform.
