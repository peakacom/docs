Experience a smooth integration process with our JIRA Cloud integration. This guide will walk you through the steps to connect JIRA Cloud with Peaka, enabling you to effortlessly manage your JIRA data within the Peaka platform.

## Prerequisites

Before you begin, you'll need to obtain an API Key from JIRA Cloud. Here's how to do it:

Follow the steps below to obtain it:

1. To generate your API Key, navigate to [Atlassian API Key Generation](https://id.atlassian.com/manage-profile/security/api-tokens) and follow the instructions to create your API Key.

<img src="https://cdn.peaka.com/integrations/jira-integration-2.png" />

2. To include your service URL, visit this [link](https://id.atlassian.com/manage-profile/link-preferences) and enter your service URL in the specified field.

<img src="https://cdn.peaka.com/integrations/jira-integration-1.png" />

## Connecting JIRA Cloud to Peaka

With your API Key and service URL at hand, you're ready to connect JIRA Cloud to Peaka. Follow these steps:

1. Go to the settings page in Peaka.

2. Click on the "Add First Connection" button to start the data source process.

3. In the pop-up that appears, choose "JIRA Cloud" as your data source.

4. Input your data source name, API Key, and service URL. The username can be any text

<img src="https://cdn.peaka.com/integrations/jira-integration-3.png" />

5. Click on "Authorize" to establish your JIRA Cloud data source.

And that's it! You have successfully integrated JIRA Cloud with Peaka. You can now access and manage your JIRA data directly within the Peaka platform. Enjoy the seamless experience!
