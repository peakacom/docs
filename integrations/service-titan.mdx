This article guides you through the process of connecting ServiceTitan to Peaka, allowing you to seamlessly access your ServiceTitan data within the Peaka platform.

## Prerequisites

Before you begin, ensure you have the following credentials from ServiceTitan:

- **Client ID**
- **Client Secret**
- **Tenant**
- **Application Key**

Follow these steps to obtain them:

1. **Client ID and Client Secret**:

   - Refer to [this article](https://developer.servicetitan.io/docs/get-going-manage-client-id-and-secret/) for instructions.

2. **Tenant**:

   - Click the gear icon ⚙️ in the top-right corner.
   - Navigate to **Integrations > API Application Access** in the left-hand menu. Your Tenant ID will be displayed on the right. Copy it.

3. **Application Key**:
   1. Log in to your [ServiceTitan Developer account](https://developer.servicetitan.io/signin/). Select either the integration or production environment, depending on which environment your app will be used in.
   2. Click the + New App button.
   3. Enter all **App details** for your custom app, including the **Tenant(s)** that will use the custom app and the necessary **API Scopes**. For additional details, refer to the [ServiceTitan Create an App API documentation](https://developer.servicetitan.io/docs/get-going-create-an-app/).
   4. Click the Create App button. You will return to the **My Apps** page.
   5. Next to your new app, click **Edit**.
   6. Copy the Application Key.

## Connecting ServiceTitan to Peaka

Now that you have your ServiceTitan Client ID, Clint Secret, Tenant and Application Key, you can connect it to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. Click the "Bring your data in" button to initiate the connection process:

3. In the modal that appears, select "ServiceTitan" as your data source.

4. Enter a name for your data source name and paste values that you before.

<img src="https://cdn.peaka.com/integrations/service-titan-1.png" />

5. Click "Authorize" to create your ServiceTitan data source.

Congratulations! You've successfully connected ServiceTitan to Peaka. You can now access and manage your ServiceTitan data seamlessly within the Peaka platform.
