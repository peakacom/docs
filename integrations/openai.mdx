This article guides you through the process of connecting OpenAI to Peaka, allowing you to seamlessly access your OpenAI data within the Peaka platform.

## Prerequisites

Before you start, you will need an API Key from OpenAI.

Follow the steps below to obtain it:

1. Go to this [link](https://platform.openai.com/account/api-keys) to create or obtain your OpenAI API Key.

2. Click the "Create new secret key" button to generate your API Key.

<img src="https://cdn.peaka.com/integrations/openai-integration-1.png" />

## Connecting OpenAI to Peaka

Now that you have your requirements, you can connect it to Peaka using the following steps:

1. Navigate to the settings page in Peaka.

2. Click the "Add First Connection" button to initiate the connection process.

3. In the modal that appears, select "OpenAI" as your connection type.

4. Enter your connection name and API Key.

<img src="https://cdn.peaka.com/integrations/openai-integration-2.png" />

5. Click "Authorize" to create your OpenAI connection.

<img src="https://cdn.peaka.com/integrations/openai-integration-3.png" />

Congratulations! You've successfully created a connection between OpenAI and Peaka. You can now access and manage your OpenAI data seamlessly within the Peaka platform.
