This article guides you through the process of connecting BigQuery (Service Account) to Peaka, allowing you to seamlessly access your BigQuery (Service Account) data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from BigQuery.

- Credentials in JSON format

Follow the steps below to obtain it:

1. Visit the [Credentials page](https://console.cloud.google.com/apis/credentials).

2. Select "+Create credentials" and then choose "Service account."

<img src="https://cdn.peaka.com/integrations/big-query-service-account-1.png" />

3. Enter a name for the service account (e.g., "Peaka BigQuery (Service Account) Read Account") and select "Create and Continue."

<img src="https://cdn.peaka.com/integrations/big-query-service-account-2.png" />

4. Ensure the service account has the following permissions:
   - BigQuery Data Editor
   - BigQuery User

<img src="https://cdn.peaka.com/integrations/big-query-service-account-3.png" />

5. You can skip the optional user access step and select "Done."

6. Your servise account will be created and click on the service account email.

<img src="https://cdn.peaka.com/integrations/big-query-service-account-4.png" />

7. Click the "Add Key" button and select "Create New Key."

<img src="https://cdn.peaka.com/integrations/big-query-service-account-5.png" />

8. Choose the "JSON" file format and click the "Create" button. This will download a JSON key file that you'll need for the integration.

<img src="https://cdn.peaka.com/integrations/big-query-service-account-6.png" />

## Connecting BigQuery (Service Account) to Peaka

Now that you have obtained the necessary integration data and credentials, you can connect BigQuery (Service Account) to Peaka using the following steps:

1. Navigate to the settings page in Peaka.

2. Click the "Add First Connection" button to initiate the connection process:

3. In the modal that appears, select "BigQuery (Service Account)" as your connection type.

4. Enter a name for your connection and upload the service account file file you downloaded.

<img src="https://cdn.peaka.com/integrations/big-query-service-account-7.png" />

5. Click "Next" to create your BigQuery (Service Account) connection.

Congratulations! You've successfully connected BigQuery (Service Account) to Peaka. You can now access and manage your BigQuery (Service Account) data seamlessly within the Peaka platform.
