This article guides you through the process of connecting Freshdesk to Peaka, allowing you to seamlessly access your Freshdesk data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from Freshdesk.

- API Key
- Domain

Follow the steps below to obtain it:

1. To create your API key in Freshdesk, please refer to this helpful [guide](https://support.freshdesk.com/support/solutions/articles/215517-how-to-find-your-api-key) that provides step-by-step instructions.

2. To create your Domain in Freshdesk, please refer to this helpful [guide](https://support.freshdesk.com/support/solutions/articles/37590-set-up-a-custom-portal-url-and-verify-dns) that provides step-by-step instructions.

## Connecting Freshdesk to Peaka

Now that you have obtained the necessary integration data and credentials, you can connect Freshdesk to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. In the side menu bar, click the "New Data Source" button to initiate the connection process.

3. In the modal that appears, select "Freshdesk" as your data source.

4. Enter a name for your data source and enter your API key and domain.

<img src="https://cdn.peaka.com/integrations/freshdesk-1.png" />

5. Click "Connect" to create your Freshdesk data source.

Congratulations! You've successfully connected Freshdesk to Peaka. You can now access and manage your Freshdesk data seamlessly within the Peaka platform.
