## Connecting Teradata to Peaka

Now that you have your Teradata API Key, you can connect it to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. Click the "Bring your data in" button to initiate the connection process:

3. In the modal that appears, select "Teradata" as your data source.

4. Paste values that you got before.
