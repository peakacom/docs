This article guides you through the process of connecting SendGrid to Peaka, allowing you to seamlessly access your SendGrid data within the Peaka platform.

## Prerequisites

Before you start, you will need an API Key from Open AI.

Follow the steps below to obtain it:

1. Go to this [link](https://www2.Sendgrid.com/app/account/integrations) to create or obtain your SendGrid API Key.

2. Click the "Create API Key" button to generate your API Key.

<img src="images/integrations/sendgrid-1.png" />

## Connecting SendGrid to Peaka

Now that you have your requirements, you can connect it to Peaka using the following steps:

1. Navigate to the settings page in Peaka.

2. Click the "Add First Connection" button to initiate the connection process.

3. In the modal that appears, select "Sendgrid" as your connection type.

4. Enter your connection name and API Key.

<img src="images/integrations/sendgrid-2.png" />

5. Click "Authorize" to create your SendGrid connection

<img src="images/integrations/sendgrid-3.png" />

Congratulations! You've successfully created a connection between SendGrid and Peaka. You can now access and manage your SendGrid data seamlessly within the Peaka platform.
