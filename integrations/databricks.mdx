This article guides you through the process of connecting Databricks to Peaka, allowing you to seamlessly access your Databricks data within the Peaka platform.

## Prerequisites

Before you start, you will need the following information from Databricks.

- URL
- PORT
- HTTP Path
- User
- Password

Follow the steps below to obtain it:

1. To obtain get URL, PORT and HTTP path, check this [article](https://docs.databricks.com/en/integrations/compute-details.html#:~:text=do%20the%20following%3A-,Log%20in%20to%20your%20Databricks%20workspace.,%2C%20Port%2C%20and%20HTTP%20path.).

2. To obtain get User and Password, check this [article](https://docs.databricks.com/api/account/accountusers/get).

## Connecting Databricks to Peaka

Now that you have obtained the necessary integration data and credentials, you can connect Databricks to Peaka using the following steps:

1. Navigate to the data model page in Peaka.

2. Click the "Bring your data" button to initiate the connection process.

3. In the modal that appears, select "Databricks" as your connection type.

4. Enter a name for your connection, input the credentials you obtained in the previous steps.

<img src="https://cdn.peaka.com/integrations/databricks-1.png" />

5. Click "Next" to create your Databricks connection.

Congratulations! You've successfully connected Databricks to Peaka. You can now access and manage your Databricks data seamlessly within the Peaka platform.
