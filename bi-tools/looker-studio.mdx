### Prerequisites:
- **A Google account**: To access **Looker Studio**, you will need a Google account. If you already have one, you can directly log into Looker Studio. If not, you can create an account in the [Looker Studio](https://lookerstudio.google.com/) page.

- A Peaka account and a Peaka Project.

- An API Key you created in your Peaka Project.

## Using Peaka as a Data Source in Looker Studio

1) Prepare your data in [Peaka](https://peaka.studio).

2) Click on the following link to add Peaka to Looker Studio as a data source: <a href="https://lookerstudio.google.com/datasources/create?connectorId=AKfycbx-8Exxvuv-IzM8m1bS5WnNcpb-a3Xri1AR5iin7qo" target="_blank">https://lookerstudio.google.com/datasources/create?connectorId=AKfycbx-8Exxvuv-IzM8m1bS5WnNcpb-a3Xri1AR5iin7qo</a>

3) To confirm the action, click on the AUTHORIZE button.
<img src="https://cdn.peaka.com/bi-tools/looker-studio-1.png" />

4) Sign in to Looker Studio using your Google account credentials and grant Google access rights by clicking on the ALLOW button.

5) Paste your Peaka API Key to the "Key" field in Looker Studio.
<img src="https://cdn.peaka.com/bi-tools/looker-studio-2.png" />

6) Select a Catalog.
<img src="https://cdn.peaka.com/bi-tools/looker-studio-3.png" />
<img src="https://cdn.peaka.com/bi-tools/looker-studio-4.png" />

7) Select a Schema (if you are working with a Peaka Table, a Peaka Query, or a BI Table, you will proceed to select a Table without being asked to select a Schema).
<img src="https://cdn.peaka.com/bi-tools/looker-studio-5.png" />
<img src="https://cdn.peaka.com/bi-tools/looker-studio-6.png" />

8) Select a Table and click on the "Connect" button.
<img src="https://cdn.peaka.com/bi-tools/looker-studio-7.png" />
<img src="https://cdn.peaka.com/bi-tools/looker-studio-8.png" />

9) Browse the fields and rename the Data Source if needed.
<img src="https://cdn.peaka.com/bi-tools/looker-studio-9.png" />

10) Add the data source to your report.

You are all set now! You have successfully added Peaka to Looker Studio as a Data Source.

You can now use your queries or tables to create insightful reports in Looker Studio.
