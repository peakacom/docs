---
title: "How to delete a project in Peaka"
description: "Get started with Project Operations: Learn how you can delete a project in Peaka."
---

Follow these steps to delete a project in Peaka:

1. Click on the three-dots button (ellipsis icon) associated with the project and click on the "Delete" button.

<img
  src="https://cdn.peaka.com/project-operations/click-3-dot.png"
  alt="open context menu of app view"
/>

2. Click on the "Delete" button on the confirmation modal.

<img
  src="https://cdn.peaka.com/project-operations/delete-app-1.png"
  alt="delete project modal view"
/>

That's it. The project will be deleted. You can see the project page on the list of projects.

<img
  src="https://cdn.peaka.com/project-operations/create-app-3.png"
  alt="list of projects"
/>
