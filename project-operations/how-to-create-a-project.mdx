---
title: "How to create a project"
description: "Get started with Project Operations: Learn how you can start a new project in Peaka."
---

Follow these steps to create a project in Peaka:

1. Click on the "New Project" button.

<img
  src="https://cdn.peaka.com/project-operations/create-app-1.png"
  alt="Peaka projects dashboard"
/>

2. Select workspace, enter the project name, description and click on the "Create Project" button.

<img
  src="https://cdn.peaka.com/project-operations/create-app-2.png"
  alt="create project modal view"
/>

That's it. The project will be created. You can see the project page on the list of projects.

<img
  src="https://cdn.peaka.com/project-operations/create-app-3.png"
  alt="list of projects"
/>
