---
title: "How to rename a project in Peaka"
description: "Get started with Project Operations: Learn how you can change the name of your Peaka project."
---

Follow these steps to rename a project in Peaka:

1. Click on the three-dots button (ellipsis icon) associated with the project and click on the "Rename" button.

<img
  src="https://cdn.peaka.com/project-operations/click-3-dot.png"
  alt="open context menu of app view"
/>

2. Enter the new project name and click on the "Change" button.

<img
  src="https://cdn.peaka.com/project-operations/rename-app-1.png"
  alt="rename project modal view"
/>

That's it. The project name will be updated. You can see the project page on the list of projects.

<img
  src="https://cdn.peaka.com/project-operations/rename-app-2.png"
  alt="list of projects"
/>
