---
title: "How to clone a project in Peaka"
description: "Get started with Project Operations: Learn how you can clone an existing project in Peaka."
---

Follow these simple steps to clone an existing project in Peaka:

1. Click on the three-dots button (ellipsis icon) associated with the project and click on the "Clone" button.

<img
  src="https://cdn.peaka.com/project-operations/click-3-dot.png"
  alt="open context menu of app view"
/>

2. Enter the project name and click on the "Clone" button.

<img
  src="https://cdn.peaka.com/project-operations/clone-app-1.png"
  alt="clone project modal view"
/>

That's it. The project will be cloned. You can see the cloned project page on the list of projects.

<img
  src="https://cdn.peaka.com/project-operations/clone-app-2.png"
  alt="list of projects"
/>
