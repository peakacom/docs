## What is the 'Set App Variable' Node?

The 'Set App Variable' node is a fundamental component in Peaka that enables you to modify the value of a selected app variable within your workflows.

## Changing the Value of App Variables

To change the value of an app variable using the 'Set App Variable' node, follow these steps:

1. Start by selecting the specific app variable you wish to modify.

2. Next, enter the new value you want to assign to the selected app variable.

That's it! Once the node executes within your workflow, it will update the value of the chosen app variable according to the provided input.

## Conclusion

The 'Set App Variable' node grants you the ability to dynamically alter app variable values, allowing for flexible data manipulation and customizations within your Peaka applications.
